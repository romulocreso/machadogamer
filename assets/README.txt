IMAGENS DO SITE — coloque seus arquivos aqui
=============================================

Para personalizar, substitua os arquivos de exemplo abaixo
(mantenha o MESMO NOME ou ajuste o caminho no index.html):

  logo.svg        -> logo/ícone do canal (também é o favicon)
  perfil.svg      -> sua foto de perfil (pode trocar por perfil.jpg/png)
  banner.svg      -> imagem de preview ao compartilhar o link
  galeria-1.svg   -> foto/clipe 1
  galeria-2.svg   -> foto/clipe 2
  galeria-3.svg   -> foto/clipe 3
  galeria-4.svg   -> foto/clipe 4

DICAS
- Se usar .jpg ou .png em vez de .svg, atualize o "src" correspondente
  no index.html (procure pelos comentários <!-- TROCAR -->).
- Formato quadrado funciona melhor para perfil e galeria.
- Pode salvar prints aqui também; me avise os nomes que eu encaixo.
